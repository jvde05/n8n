# Arka plan muzigi (opsiyonel)

Buraya `bgm.mp3` adinda telifsiz bir muzik koyarsan render sirasinda
arka plana dusuk sesle eklenir. Dosya yoksa video sadece seslendirme ile uretilir.

Ucretsiz telifsiz muzik kaynaklari:
- YouTube Audio Library (Studio > Ses kitapligi)
- https://pixabay.com/music/
- https://incompetech.com/
